About this site
---------------

Every document is tagged with one or more labels. A document can appear under multiple tags without being duplicated - it is still one file. External links are included in the same index so there is one searchable front door for everything regardless of where content actually lives.

**Tag reference**

.. list-table::
   :header-rows: 1
   :widths: 20 80

   * - Tag
     - What it covers
   * - ``getting-started``
     - Setup, onboarding, first steps
   * - ``guides``
     - Step-by-step how-to guides
   * - ``procedures``
     - Operational procedures and processes
   * - ``runbooks``
     - Step-by-step operational runbooks
   * - ``templates``
     - Reusable document templates
   * - ``reference``
     - Architecture, system overviews, lookup material
